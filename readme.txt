This project is a very heavily modified gazelle source.

It requires the matching modified ocelot tracker. https://github.com/mifune/ocelot


Major changes form the gazelle source:

only one torrent per torrent group.
big expansion of bbcode.
many improvements to forum tools.
torrent staff review system.
extended tag system (synonyms).
user awards and badges system.
numerous tweaks and improvements.
